malt.data.collections.linear_alkanes

malt.data.collections.linear_alkanes(max_carbon=10)[source]

A toy dataset with linear alkanes from 1 to max_carbon carbons.

Parameters

max_carbon (int) – Maximum number of carbons in the molecules generated.

Returns

A dataset containing alanes.

Return type

Dataset

Examples

dataset = count_carbons(10)
